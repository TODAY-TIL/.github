## About

**TODAY-TIL**은 매일 배운 개발 지식을 기록하는 TIL 아카이브입니다.

프론트엔드 개발을 중심으로 JavaScript, React, CSS, Git, Web, 에러 해결 과정 등을 정리합니다.

단순히 공부한 내용을 저장하는 것이 아니라,  
나중에 다시 봐도 이해할 수 있도록 **개념, 예제, 문제 상황, 해결 과정**을 함께 기록합니다.
